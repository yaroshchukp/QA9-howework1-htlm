Изученные теги

<html>

Тег <html> - контейнер для содержимого страницы.

Пример: <html>"Вся информация на странице"</html>

<head>

Тег <head> содержит служебную информацию. Здесь подключаются стили, указывается заголовок страницы, подключаются метатеги.

Пример: <head><title>Document</title></head>

<body>

Тег <body> содержит контент — всё, что отображается в браузере.

Пример: <body>"Весь контент страницы"</body>

<div>

Тег <div> используется как контейнер для HTML-элементов.

Пример: <div>Все, что угодно для создания элементов - классы, блоки, и т.д.</div>

<a>

Тег <a> используется для создания ссылок.

Пример: <a href="https://google.com">Google</a>

<h1> - <h6>

Теги <h1>-<h6> используются для создания заголовков разного уровня.

Пример: <h1>Заголовок первого уровня</h1>

<p>

Тег <p> используется для создания абзацев текста.

Пример: <p>Абзац текста: Lorem ipsum dolor sit amet consectetur adipisicing elit. Veniam repellendus optio asperiores architecto maiores. Iste necessitatibus voluptas, distinctio eos adipisci nisi reiciendis cum error ullam. Minus recusandae ipsa voluptas vitae.</p>

<img>

Тег <img> используется для добавления изображений.

Пример: <img src="https://iapn.kz/upload/iblock/2ce/2ce940854e9a109750531a955440acb3.jpg" alt="Чорный Кошк!">

<link>

Тег <link> подключает CSS-файлы.

Пример: <link rel="stylesheet" href="style.css"> - подключение CSS-файла

<br>

Тег <br> переносит текст на другую строку, не начиная нового абзаца

Пример: <p>1ая строка текста.<br>2ая строка текста.</p>
